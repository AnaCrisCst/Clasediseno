Lo que aprendimos

Aprendimos básicos del coding en CodeAcademy, y en clase usamos GitHub + VisualCode combinados para hacer esta paginita.

También aprendimos que el index.html funciona para subirlo todo a internet.

En GitHub, la sección "README" funciona para pegar el link.

Los "comits" son los cambios que has hecho a tu página, es como el historial.

Para elegir una plantilla apta para la página sobre la que trabajaremos todo el semestre, Edu nos dijo que consideráramos algunas cositas importantes:

-Buscar webs limpias o sin muchos elementos.

-Que el formato tuviera opción para agregar subpáginas.

Para agregar imagenes al sitio web, debemos descargarlas de internet, agregarlas a la carpeta de "imagenes" ubicada dentro de la carpeta "htmlup-phantom", y anotarla tal cual se llama + su extensión ".jpg"
